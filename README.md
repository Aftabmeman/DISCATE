
# Mentur AI | Elite Academic Mentorship

Mentur AI is an advanced academic mentorship platform that leverages AI to generate personalized assessments and provide deep evaluation of student work.

## GitHub Terminal Instructions

Follow these commands to push your code to your GitHub repository:

```bash
# 1. Initialize the git repository
git init

# 2. Add all files to the staging area
git add .

# 3. Create a commit with a message
git commit -m "Initial commit - Elite Academic Mentorship"

# 4. Set the default branch to main
git branch -M main

# 5. Add your remote repository URL
# Replace <URL> with your actual GitHub repository URL
git remote add origin <YOUR_GITHUB_REPO_URL>

# 6. Push the code to your repository
git push -u origin main
```

## Core Features
- **Sequential Assessment Wizard**: Generate MCQs, Flashcards, and Essays from study materials.
- **Dynamic Limits**: 1 to 5 questions for Essay practice; 10 to 30 for objective modes.
- **Professional Exam Report Card**: Animated Score Circle with deep metrics (Grammar, Depth, Relevancy).
- **Regional Mix Support**: Mentorship in 10 language styles (Hinglish, Marathish, etc.) with a "Baval" tone.
- **Secure Architecture**: Built with Next.js, Firebase Auth, and Firestore user silos.
